# hotaruika-watch

新潟県・富山県のホタルイカ身投げ・掬い情報を自動収集し、スマホで見やすい掲示板形式で表示するシステムです。

🔗 https://afghani420.github.io/hotaruika-watch/

---

## ツール一覧

### 🦑 ホタルイカ情報掲示板
https://afghani420.github.io/hotaruika-watch/

ホタルイカの身投げ・掬い情報をリアルタイム収集。Serper.dev + Claude AIで自動更新（2時間毎）。

### 🔍 AliExpress多言語検索
https://afghani420.github.io/hotaruika-watch/tools/ali-search/

日本語でキーワードを入力すると、英語・中国語に自動翻訳してAliExpressの検索リンクを3言語同時生成するモバイル向けツール。検索履歴（最大12件）付き。
